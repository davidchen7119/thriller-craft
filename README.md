# Thriller-Craft

悬疑/推理/惊悚小说全流程创作系统 —— QoderWork Skill

## 简介

Thriller-Craft 是一个为 QoderWork 设计的结构化创作技能，覆盖悬疑/推理/惊悚小说从构思到成稿的完整流程。系统支持 9 大子类型，提供 6 阶段创作管线，帮助 AI 助手按照专业方法论完成高质量的类型小说创作。

## 支持的子类型

- 本格推理（Whodunit）
- 社会派推理
- 心理悬疑
- 密室/封闭圈
- 惊悚/悬念
- 硬汉派/冷硬派
- 倒叙推理
- 日常推理
- 法律/医学/历史等专业推理

## 6 阶段创作管线

1. **项目立项** — 确定类型、篇幅、风格基调
2. **主题设计** — Egri 方法构建前提
3. **诡计与线索设计** — 核心谜题、线索布局、红鲱鱼
4. **人物系统** — 嫌疑人矩阵、动机网络
5. **叙事结构** — 章节规划、信息节奏
6. **写作与修改** — 逐章创作、质量评估

## 使用方式

在 QoderWork 中安装本 Skill 后，直接描述你的创作需求即可。例如：

> "我想写一个短篇本格推理故事，8000字以内，用不在场证明诡计"

Skill 会自动引导完成从设计到成稿的全流程。
